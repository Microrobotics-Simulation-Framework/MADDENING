**Subtasks**

- [ ] XOR-delta optional path (off by default)
- [ ] Benchmark: 256³ LBM 60fps consecutive-frame compression ratio (target ~10:1 with zstd alone)

## Cross-cutting risks

1. **Pencil correctness under coupling.** Sharded stencil inside a coupling group is the convergence edge case nobody has hit yet. Plan a specific test for sharded-Heat ↔ sharded-LBM on the same pencil axis.
2. **Multi-device gradient audit.** `ppermute` is differentiable but the test infrastructure isn't there. Treat it as an explicit subtask of #1, not a postscript.
3. **Three simultaneous breaking contracts (#1, #3, #4).** Lab arrival timing means new students write against whatever is current. Migration guide written before week 4; ship #4 as warnings.
4. **`replace_node` × `static_data` × sharding** is a three-way interaction. Spec in `DESIGN.md` before #3 lands.
